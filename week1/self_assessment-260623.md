## ① สรุปเนื้อหาที่เรียน — ฉันได้เรียนรู้อะไรในสัปดาห์นี้?

ได้เรียนรู้เกี่ยสวกับ Pipeline ในการพัฒนาเกม โดยเรียนถึงความหมายของ Pipeline (อะไรคือ Pipeline), เปรียบเทียบกับ Produciton Pipeline ในการทำหนัง, และ ประโยชน์ของการเข้าใจ Pipeline รวมถึงการได้ดูตัวอย่างของ Game Pipeline
อีกทั้งยังได้เรียนรู้เกี่ยวกับตำแหน่งงาน/หน้าที่ในอุตสาหกรรมเกม จนได้เข้าใจที่จะค้นหาตำแหน่งงานที่จะตรงกับความต้องการของตนเอง

## ② สรุปการฝึกปฏิบัติ — ฉันได้ทดลองและฝึกเขียนโปรแกรมอะไรบ้าง?

ใน Lab ได้ set up environment ในการทำงาน ได้ใช้ command line ในการโหลดโปรแกรม และยังได้ใช้ Git เพื่อทำ version control แบบเบื้องต้น (git init + git push)

## ③ ปัญหา / ข้อสงสัย — สิ่งที่ยังไม่เข้าใจหรืออยากสำรวจต่อ

เนื่องจากเคยได้ลองใช้ git มาบ้างจึงค่อนข้างเข้าใจการใช้งานเบื้องต้น แต่ยังไม่เคยได้ใช้งาน git กับงานสเกลใหญ่ร่วมกับคนหลายๆคน ซึ่งจะส่งผลต่อการ push แล้วพอได้เห็นมาคร่าวๆเช่นเรื่องการแตก branch การ merge งานเรากับของคนอื่น ซึ่งมองว่ามีความซับซ้อนกว่าพอสมควรจึงอยากเรียนรู้ในส่วนนี้เพิ่มเติม
